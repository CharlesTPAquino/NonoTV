# Guía de Modelos de IA Gratuitos por Tarefa

## Modelos Disponíveis
- **Claude 3 Haiku** - Código, análise, velocidade
- **GPT-4o Mini** - Versatilidade, raciocínio
- **Gemini 1.5 Flash** - Multimodal, contexto longo

---

## 🚀 Proposta por Tipo de Tarefa

### 💻 **Código / Programação**
**Recomendado**: Claude 3 Haiku

> "Use Claude 3 Haiku para esta tarefa de código. Ele é excelente em análise de código, refatoração, debug e explicações técnicas."

---

### 📝 **Escrita / Documentação**
**Recomendado**: Claude 3 Haiku ou GPT-4o Mini

> "Use Claude 3 Haiku para escrita técnica ou GPT-4o Mini para textos criativos. Ambos produzem conteúdo natural e bem estruturado."

---

### 🔍 **Pesquisa / Análise de Arquivos**
**Recomendado**: Gemini 1.5 Flash

> "Use Gemini 1.5 Flash para esta análise. Seu contexto longo (1M tokens) permite processar documentos extensos de uma vez."

---

### 🧮 **Raciocínio / Resolução de Problemas**
**Recomendado**: GPT-4o Mini

> "Use GPT-4o Mini para esta tarefa de raciocínio. Ele tem excelente performance em matemática e lógica."

---

### 🎨 **Multimodal (Imagens + Texto)**
**Recomendado**: Gemini 1.5 Flash

> "Use Gemini 1.5 Flash para processar imagens e texto juntos. É nativo para multimodal."

---

### ⚡ **Tarefas Rápidas / Simples**
**Recomendado**: Claude 3 Haiku

> "Use Claude 3 Haiku para esta tarefa simples. É o mais rápido (~300ms) e econômico."

---

## 📋 Prompt Padrão

Quandoavorar qual modelo usar, completar:

```
Tipo de tarefa: [código/escrita/pesquisa/raciocínio/multimodal/rápida]
Complexidade: [baixa/média/alta]
Urgência: [baixa/alta]

Modelo recomendado: [Claude Haiku / GPT-4o Mini / Gemini Flash]
Justificativa: [1-2 frases]
```

---

## 🎯 Decisão Rápida

| Necessidade | Modelo |
|-------------|--------|
| Precisa de código limpo | Claude Haiku |
| Precisa de velocidade | Claude Haiku |
| Precisa de contexto longo | Gemini Flash |
| Precisa de multimodal | Gemini Flash |
| Precisa de raciocínio | GPT-4o Mini |
| Precisa de tudo-arredondado | GPT-4o Mini |
